Academic Performance Prediction — Project Report

A machine learning study analyzing demographic, behavioral, socioeconomic, and well-being features to predict semester GPA of international university students.

View Notebook & Results Download Notebook

About the Project

This project examines academic performance among international students using a holistic dataset of 25 demographic, socioeconomic, behavioral, and psychological variables. Unlike traditional student-performance datasets that rely mainly on exam scores and basic demographics, this study incorporates work hours, stress, sleep patterns, financial support, social support, and lifestyle indicators to build a more realistic model of student success.

The goal is to predict semester GPA accurately and identify the strongest contributors to academic performance, enabling early detection of at-risk students and supporting targeted interventions.

Data & Preprocessing

  • 25-feature dataset combining academic, behavioral, socioeconomic & well-being variables
  • Label encoding for categorical variables
  • Standardization of all numerical features
  • 75/25 train-test split with reproducible random state
  • Pipeline includes encoders, scaler, and final model (ElasticNet) saved via joblib

Models Evaluated

  • Linear Regression, Ridge, Lasso, ElasticNet, Bayesian Ridge
  • Support Vector Regression (RBF)
  • Random Forest, Extra Trees
  • Gradient Boosting, XGBoost, CatBoost, LightGBM
  • 5-fold cross-validation used for RMSE & R² comparison

Key Results

ElasticNet achieved the best performance with a cross-validated R² of 0.8625 and RMSE of 0.1595, outperforming all other linear, regularized, and ensemble models. Tree-based models such as Random Forest, XGBoost, and Extra Trees showed higher error, indicating that the dataset is predominantly linear in structure.

Strongest predictors included study hours, attendance, sleep duration, social support, and previous semester GPA.
